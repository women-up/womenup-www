# Women Up!

Oddolna inicjatywa spoleczna z Podlasia. Laczymy kobiety roznych statusow spolecznych.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix)
- Supabase
- React Router DOM v6
- Framer Motion

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
