Build a complete, responsive website for "Women Up!" — a Polish women's social initiative association from Podlasie region. The site should be informational, warm, and culturally authentic, with a distinctive Slavic folk art cross-stitch aesthetic.

## BRAND IDENTITY AND DESIGN SYSTEM

### Color Palette (Tailwind config)
Use these exact colors throughout:
- Primary red: #C41A2C (deep Polish red — use for headings, buttons, accents, logo)
- Primary dark: #9E1523 (hover states)
- Primary light: #E8344A (subtle accents)
- Background: #F5F0E8 (warm cream — main page background, NOT white)
- Background alt: #EDE7DB (alternating sections)
- Surface: #FFFFFF (cards, forms, modals)
- Text primary: #2D2016 (dark warm brown, NOT black)
- Text secondary: #6B5E52 (muted brown)
- Border: #D4CBBC (warm light border)

### Typography
- Headings: Cormorant Garamond from Google Fonts (serif, elegant, cultural). Weights: 400, 600, 700. Add letter-spacing: 0.02em on H1.
- Body text: Inter from Google Fonts (clean sans-serif). Weights: 400, 500, 600.
- Scale: H1=56px, H2=40px, H3=28px, H4=22px, Body=16px, Small=14px, Caption=12px.

### Slavic Cross-Stitch Design Motif (CRITICAL — this defines the visual identity)
The brand uses a Slavic folk cross-stitch diamond pattern. Incorporate these decorative elements:

1. Page top border: A thin repeating pixel/cross-stitch pattern strip (4px height) at the very top of the page in primary red. Create this with CSS repeating gradient or a tiny SVG pattern of small squares.

2. Section dividers: Between major sections, use a horizontal line with a small diamond character centered. Style: thin line #D4CBBC, diamond in #C41A2C. Like this: a thin line, then a diamond symbol, then a thin line.

3. Hero background pattern: A very subtle (opacity: 0.03-0.05) repeating diamond/cross-stitch grid pattern behind the hero section. Use CSS background pattern or SVG.

4. Card corner decorations: Optional tiny pixel squares (4x4px) in the top-left corner of cards, in primary red with low opacity.

5. Footer pattern: Repeating cross-stitch border along the top of the footer.

### Buttons
- Primary: bg #C41A2C text white, rounded-sm (2px, NOT rounded-full), px-8 py-3, hover: bg #9E1523, uppercase tracking-wider text-sm font-semibold
- Secondary: border-2 border #C41A2C text #C41A2C, transparent bg, same shape
- Ghost: text #C41A2C, underline on hover

### Cards
White background, border border-#D4CBBC, rounded-sm, p-6, hover: shadow-md + border-#C41A2C transition

### General rules
- Border radius: use rounded-sm (2px) everywhere. NO rounded-full or rounded-xl. The aesthetic is geometric, not bubbly.
- Shadows: minimal. Use shadow-sm or none. Warmth comes from color, not depth.
- Everything should feel warm, cultural, handcrafted — NOT corporate or generic SaaS.
- Cream backgrounds, NOT white. The warmth of the cream (#F5F0E8) is essential to the brand.
- NO stock photo placeholders with blue/purple gradients. Use #F5F0E8 cream backgrounds with red accents.

## RESPONSIVE DESIGN

- Mobile-first approach
- Navbar: collapses to hamburger on mobile
- Hero: reduce H1 to 36px on mobile, stack buttons vertically
- Card grids: 3 cols then 2 cols then 1 col
- Sections: reduce vertical padding on mobile
- Font sizes: scale down approximately 20% on mobile
- Touch-friendly: minimum 44px tap targets

## IMPORTANT IMPLEMENTATION NOTES

1. Use React Router for multi-page navigation
2. All text content is in POLISH — do not translate to English
3. The overall feeling should be: warm, cultural, authentic, community-focused — NOT corporate, NOT startup-y
4. Cream backgrounds, NOT white. The warmth of the cream (#F5F0E8) is essential to the brand.
5. Red (#C41A2C) is used for accents, headings, buttons — NOT for large background areas (except the CTA band)
6. Typography hierarchy is important: Cormorant Garamond for all headings (elegant serif), Inter for body text
7. The cross-stitch/pixel pattern decorations are subtle but present — they tie everything to the Slavic folk art identity
8. Sharp corners (rounded-sm, 2px) — this is geometric, not soft/bubbly
9. Include a favicon — a small red diamond shape on cream background
10. Add Google Fonts imports for Cormorant Garamond and Inter
11. All images are placeholders for now — use cream-colored boxes with subtle diamond pattern or Lucide icons
12. Footer should feel substantial and warm, not minimal
